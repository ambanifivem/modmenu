- ❌ **Menu Does Not Open After Pressing F5**
  The overlay may fail to load if Ambani Fivem Mod Menu was launched before FiveM finished initializing. **Solution:** Increase `injection_delay` to `5000ms` in `settings.cfg` and relaunch the loader after FiveM is fully loaded.

- ❌ **Crash on Injection**
  This usually happens when antivirus software intercepts the injection process mid-execution. **Solution:** Add the entire `C:\Desktop\AmbaniMenu\` folder to your antivirus exclusion list and disable real-time protection temporarily during launch.

- ❌ **Overlay Flickering or Not Rendering**
  Display driver incompatibilities can cause the Ambani Fivem Mod Menu overlay to flicker. **Solution:** Update your GPU drivers to the latest version and ensure no other overlay software like Discord overlay is running simultaneously.

- ❌ **Features Not Working on Specific Servers**
  Some FiveM servers run custom anti-cheat scripts that block certain functions. **Solution:** Enable `stealth_mode` and `heartbeat_spoof` in the configuration, then reconnect to the server with Ambani Fivem Mod Menu reloaded.

- ❌ **Auto Updater Stuck on Checking**
